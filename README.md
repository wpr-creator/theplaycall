# The Play Call

Repo-ready HTML package for:

https://github.com/wpr-creator/theplaycall

Upload these files at the root of the repository:

- index.html
- new_deal.html
- highway.html
- tarp.html
- cares.html
- images/

All image paths use relative links such as:

images/migrant_mother.jpg

Do not use a leading slash before images.
